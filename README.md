# webxdc-code-playground

Simple [webxdc](https://webxdc.org) that allows you to code, preview and build a webxdc or just test some code, no setup needed, just open and code something.

Also running as a [PWA](https://en.wikipedia.org/wiki/Progressive_web_app) at https://webxdc-code-playground.onrender.com/

## development

### install dependencies

```sh
pnpm install
```

### test in browser

```sh
pnpm run dev
```

### build

```sh
pnpm run build
```

To build with [eruda](https://eruda.liriliri.io/):

```sh
NODE_ENV=debug pnpm run build
```
